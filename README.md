📞 SmartDialer App
Overview

This app allows users to make direct phone calls by entering a number, without opening the default dialer interface.

Features
Direct calling functionality
Simple UI
Uses phone intent
Technologies Used
Java
XML
Android Intent System
Project Structure
DialerActivity.java → Handles call intent
activity_dialer.xml → UI for number input
How It Works
User enters phone number
Clicks "Call"
App triggers direct call using intent
Future Improvements
